# Python Study Tracker

A simple command-line study tracker built with Python.

This program allows users to log study sessions by subject and track how long they studied. The goal of this project is to practice Python programming while creating a useful productivity tool.

---

## Features

* Log study sessions by subject
* Track study duration
* Store study history
* Simple command-line interface
* Lightweight and easy to run

---

## Example Usage

Example interaction:

```
Enter subject studied: Physics
How many minutes did you study? 45

Session logged!
Subject: Physics
Duration: 45 minutes
```

## How to Run

1. Install Python
2. Clone the repository
3. Run the script

```
python study_tracker.py
```
